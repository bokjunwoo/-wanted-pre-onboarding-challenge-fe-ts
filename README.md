## ⚙️ 리팩토링 목적

- 기존에는 단기간 내에 구현해야 했기 때문에 예상치 못한 오류, 코드의 가독성과 유지보수의 어려움, 초기 로딩 속도 문제 등이 발생하였습니다. 이를 해결하고 검색 엔진 최적화(SEO)를 위해 React 코드를 서버사이드 렌더링(SSR)할 수 있도록 변환하는 리팩토링 작업을 시작하게 되었습니다.
